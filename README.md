# Shubodaya Kumar | Portfolio Blog

Live site (GitHub Pages): https://shubodaya.github.io/portfolio-blog/

## What This Is
This is my personal portfolio blog. I use it to publish write-ups on networking, cybersecurity, automation, and lab work, plus updates about what I am building and learning.

## What You'll Find Here
1. Blog posts showcasing real-world networking, security, and automation work.
2. Project breakdowns, labs, and troubleshooting notes.
3. Career updates, certifications, and learning milestones.
4. Tags and categories to quickly explore topics.

## Deployment
This repo deploys to GitHub Pages using `.github/workflows/deploy-pages.yml` on every push to `main`.

## Content Publishing
Publish posts by committing Markdown files to `src/content/posts` (locally or through GitHub UI edits/PRs).
